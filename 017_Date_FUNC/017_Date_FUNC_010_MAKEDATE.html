日期函数

新建日期/时间

Oracle 


-- 可以通过 年*10000 + 月*100 + 日 的方式， 来进行处理.

SQL> ALTER session SET nls_date_format='yyyy-mm-dd';

会话已更改。

SQL> 
SQL> SELECT TO_DATE(TO_CHAR( 2013*10000 + 6*100 + 20 ), 'YYYYMMDD') AS MAKEDATE FROM dual;

MAKEDATE
----------
2013-06-20
DB2
SQL Server 


通过  DATEADD 的方式， 针对 1900-01-01 的基础上， 作日期增加的处理.


1> SELECT
2>   DATEADD(
3>      dd,
4>      11 -1,
5>      DATEADD(
6>              mm,
7>              3 -1,
8>              DATEADD(yy, 2013 - 1900, 0))) RESULT;
9> go
RESULT
-----------------------
2013-03-11 00:00:00.000

(1 行受影响)
MySQL 


mysql> SELECT MAKEDATE (2010, 100) A;
+------------+
| A          |
+------------+
| 2010-04-10 |
+------------+
1 row in set (0.00 sec)

mysql>
mysql> SELECT MAKETIME (10, 20, 30) A;
+----------+
| A        |
+----------+
| 10:20:30 |
+----------+
1 row in set (0.00 sec)
SQLite 


使用函数进行处理，格式固定

1.date(timestring, modifier, modifier, ...) 
2.time(timestring, modifier, modifier, ...) 
3.datetime(timestring, modifier, modifier, ...) 
4.julianday(timestring, modifier, modifier, ...) 
5.strftime(format, timestring, modifier, modifier, ...)

COUNT_BIG

Oracle
DB2
SQL Server 


返回组中的项数。COUNT_BIG 的用法与 COUNT 函数类似。
两个函数唯一的差别是它们的返回值。
COUNT_BIG 始终返回 bigint 数据类型值。
COUNT 始终返回 int 数据类型值。

1> SELECT
2>   sale_item,
3>   COUNT_BIG(sale_money)
4> FROM
5>   sale_report
6> GROUP BY
7>   sale_item
8> go
sale_item
--------- --------------------
A                          365
C                          365
B                          365

SET FMTONLY { ON | OFF } 

备注
当 SET FMTONLY 为 ON 时，不会因请求而对任何行进行处理或将其发送到客户端。
SET FMTONLY 的设置是在执行或运行时设置，而不是在分析时设置。

默认情况下：
1> SELECT 'Hello World'
2> GO
-----------
Hello World
(1 行受影响)

1> SET FMTONLY ON
2> GO

1> SELECT 'Hello World'
2> GO

-----------

(0 行受影响)

1> SELECT 'Hello World' a b c
2> GO
消息 102，级别 15，状态 1，服务器 HOME-BED592453C\SQLEXPRESS，第 1 行
'b' 附近有语法错误。

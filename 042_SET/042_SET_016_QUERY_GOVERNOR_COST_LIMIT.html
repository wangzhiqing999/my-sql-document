查询开销设置

Oracle
DB2
SQL Server 


SET QUERY_GOVERNOR_COST_LIMIT value

参数
value
数值或整数值，用于指定可以运行查询的最长时间。
如果为该选项指定一个非零、非负的数值，
则查询调控器将不允许执行估计开销超过该值的查询。如果指定此选项为 0（默认），将关闭查询调控器，并且允许所有查询无限期运行。

“查询开销”是指在特定硬件配置中完成查询所需的估计占用时间（秒）。

备注
仅限于在当前连接中使用 SET QUERY_GOVERNOR_COST_LIMIT，
且只在当前连接期间有效。使用 sp_configure 的查询调控器开销限制选项可以更改服务器范围的查询调控器开销限定值。
SET QUERY_GOVERNOR_COST_LIMIT 的设置是在执行或运行时设置的，而不是在分析时设置的。

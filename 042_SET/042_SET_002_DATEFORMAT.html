数据库会话的基本参数设置 SET

设置输入的年月日顺序

Oracle
DB2
SQL Server 


SET DATEFORMAT { format | @format_var } 
参数
format | @format_var
日期部分的顺序。可以是 Unicode，或者是转换为 Unicode 的 DBCS。有效参数包括 mdy、dmy、ymd、ydm、myd 和 dym。美国英语默认值是 mdy。
备注
该设置仅用在将字符串转换为日期值时的解释中。它不影响日期值的显示。
SET DATEFORMAT 的设置是在执行或运行时设置，而不是在分析时设置。
SET DATEFORMAT 将覆盖 SET LANGUAGE 的隐式日期格式设置。

1> DECLARE @datevar DATETIME;
2>
3> SET DATEFORMAT mdy;
4> SET @datevar = '12/31/1998';
5> PRINT @datevar
6>
7> SET DATEFORMAT ydm;
8> SET @datevar = '1998/31/12';
9> PRINT @datevar
10>
11> SET DATEFORMAT ymd;
12> SET @datevar = '1998/12/31';
13> PRINT @datevar
14> GO

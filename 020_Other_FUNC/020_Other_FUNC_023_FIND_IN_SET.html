FIND_IN_SET(str,strlist) 
假如字符串str 在由N 子链组成的字符串列表strlist 中， 则返回值的范围在 1 到 N 之间 。
一个字符串列表就是一个由一些被‘,’符号分开的自链组成的字符串。如果第一个参数是一个常数字符串，
而第二个是type SET列，则   FIND_IN_SET() 函数被优化，使用比特计算。

如果str不在strlist 或strlist 为空字符串，则返回值为 0 。
如任意一个参数为NULL，则返回值为 NULL。 这个函数在第一个参数包含一个逗号(‘,’)时将无法正常运行。


mysql> SELECT FIND_IN_SET('b','a,b,c,d') A;
+---+
| A |
+---+
| 2 |
+---+
1 row in set (0.00 sec)

取得字段所占用字节数 VSIZE / DATALENGTH

Oracle 


SQL> SELECT
  2    VSIZE(SYSDATE) A,
  3    VSIZE(user) B
  4  FROM
  5    dual;

         A          B
---------- ----------
         7          2
DB2
SQL Server 


1> SELECT
2>   DATALENGTH(GETDATE()) A,
3>   DATALENGTH(user) B
4> go
A           B
----------- -----------
          8           6

取指定位置的数据 ELT

Oracle
DB2
SQL Server
MySQL 


ELT(N,str1,str2,str3,...) 
若N = 1，则返回值为  str1 ，若N = 2，则返回值为 str2 ，以此类推。   
若N 小于1或大于参数的数目，则返回值为 NULL 。 ELT() 是  FIELD()的补数。


mysql> SELECT ELT(1, 'ej', 'Heja', 'hej', 'foo') A;
+------+
| A    |
+------+
| ej   |
+------+
1 row in set (0.00 sec)

mysql>
mysql> SELECT ELT(4, 'ej', 'Heja', 'hej', 'foo') A;
+------+
| A    |
+------+
| foo  |
+------+
1 row in set (0.00 sec)

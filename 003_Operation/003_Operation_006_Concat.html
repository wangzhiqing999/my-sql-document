运算符号

字符串串联运算符

Oracle 


SQL> SELECT
  2    CONCAT ('Good ', 'Morning!') A,
  3    'Good ' ||  'Morning!'  B
  4  FROM
  5    dual;

A                          B
-------------------------- --------------------------
Good Morning!              Good Morning!
DB2 


和 Oracle 一样。

db2 => SELECT
db2 (cont.) =>   CONCAT('Good ', 'Morning!') A,
db2 (cont.) =>   'Good ' || 'Morning!'  B
db2 (cont.) => FROM
db2 (cont.) =>   SYSIBM.SYSDUMMY1;

A             B
------------- -------------
Good Morning! Good Morning!

  1 条记录已选择。
SQL Server 


加号 (+) 是字符串串联运算符，可以用它将字符串串联起来。
其他所有字符串操作都使用字符串函数进行处理。

SELECT 'Good ' + 'Morning!'  B
Go

B
Good Morning!

(1 行受影响)
MySQL 


MySQL 只能 concat.
mysql> select CONCAT('Good', 'Morning!') A;
+--------------+
| A            |
+--------------+
| GoodMorning! |
+--------------+
1 row in set (0.00 sec)

sqlite> select 'Good' || ' Morning!' A;

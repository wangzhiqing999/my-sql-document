=（等于）
相当于 指定 属性条件进行查询.
> db.TestCompare.find( {"a":3} );
{ "_id" : ObjectId("52550e37f62149b6e6e2da2c"), "a" : 3, "b" : 6, "c" : 9 }


> （大于）
使用 "$gt" 进行处理.
> db.TestCompare.find( {"a":  {"$gt" : 3} } );
{ "_id" : ObjectId("52550e37f62149b6e6e2da2d"), "a" : 4, "b" : 8, "c" : 12 }
{ "_id" : ObjectId("52550e39f62149b6e6e2da2e"), "a" : 5, "b" : 10, "c" : 15 }


< （小于） 
使用 "$lt" 进行处理.
> db.TestCompare.find( {"a":  {"$lt" : 3} } );
{ "_id" : ObjectId("52550e37f62149b6e6e2da2a"), "a" : 1, "b" : 2, "c" : 3 }
{ "_id" : ObjectId("52550e37f62149b6e6e2da2b"), "a" : 2, "b" : 4, "c" : 6 }


>=（大于等于）
使用 "$gte" 进行处理.
> db.TestCompare.find( {"a":  {"$gte" : 3} } );
{ "_id" : ObjectId("52550e37f62149b6e6e2da2c"), "a" : 3, "b" : 6, "c" : 9 }
{ "_id" : ObjectId("52550e37f62149b6e6e2da2d"), "a" : 4, "b" : 8, "c" : 12 }
{ "_id" : ObjectId("52550e39f62149b6e6e2da2e"), "a" : 5, "b" : 10, "c" : 15 }


<=（小于等于）
使用 "$lte" 进行处理.
> db.TestCompare.find( {"a":  {"$lte" : 3} } );
{ "_id" : ObjectId("52550e37f62149b6e6e2da2a"), "a" : 1, "b" : 2, "c" : 3 }
{ "_id" : ObjectId("52550e37f62149b6e6e2da2b"), "a" : 2, "b" : 4, "c" : 6 }
{ "_id" : ObjectId("52550e37f62149b6e6e2da2c"), "a" : 3, "b" : 6, "c" : 9 }


<>（不等于）
使用 "$ne"  进行处理.

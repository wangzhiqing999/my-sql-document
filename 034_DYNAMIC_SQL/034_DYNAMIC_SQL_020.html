动态SQL

小结

Oracle 


EXECUTE IMMEDIATE 用于执行 不关心返回值的SQL

DBMS_SQL 用于执行 查询，需要获取数据的，必须要按游标来处理。
DB2
SQL Server 


EXECUTE用于执行 不关心返回值的SQL

SP_EXECUTESQL 可以穿入参数，指定输出。
MySQL 


主要以下面的方式 动态执行 SQL

PREPARE sl FROM @sql;
EXECUTE sl;
DEALLOCATE PREPARE sl;
SQLite
PostgreSQL 


动态 SQL 语法

PostgreSQL 8.1  文档.
EXECUTE command-string [ INTO target ];


PostgreSQL 9.2  文档.
EXECUTE command-string [ INTO [STRICT] target ] [ USING expression [, ... ] ];

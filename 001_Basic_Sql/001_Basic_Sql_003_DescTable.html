某些情况下， 需要显示 表的 数据库存储引擎， 以及字符编码等信息的情况下， 使用 show create table 语句。


mysql> show create table  test_user_orderby;
+-------------------+-----------------------------------------------------------
------------------------------------------------------------+
| Table             | Create Table
                                                            |
+-------------------+-----------------------------------------------------------
------------------------------------------------------------+
| test_user_orderby | CREATE TABLE `test_user_orderby` (
  `chinese_number` varchar(30) DEFAULT NULL
) ENGINE=InnoDB DEFAULT CHARSET=latin1 |
+-------------------+-----------------------------------------------------------
------------------------------------------------------------+
1 row in set (0.00 sec)


从上面的输出结果， 可以看出， 数据库存储引擎为 InnoDB  默认字符集为 latin1




下面是查询数据库的情况.

mysql> show create database test;
+----------+-----------------------------------------------------------------+
| Database | Create Database                                                 |
+----------+-----------------------------------------------------------------+
| test     | CREATE DATABASE `test` /*!40100 DEFAULT CHARACTER SET latin1 */ |
+----------+-----------------------------------------------------------------+
1 row in set (0.00 sec)

要求， 把  TEST  替换为  TOP.
但是仅仅替换 前 3行。
Oracle 


update 
  testTopN
set 
  val = replace(val, 'TEST', 'TOP')
WHERE
  rownum <=3;

按逆序更新 id 最大的3个.

update 
  testTopN
set 
  val = replace(val, 'TEST', 'MAX')
ORDER BY
  id DESC
LIMIT 3;

安全函数

指明当前的用户登录是否是指定的服务器角色的成员 IS_SRVROLEMEMBER

Oracle
DB2
SQL Server 


login 的数据类型为 sysname，默认值为 NULL。如果未指定，那么使用当前用户的登录帐户。
0 login 不是 role 的成员。 
1 login 是 role 的成员。 
NULL role 或 login 是无效的。


1> SELECT
2>   IS_SRVROLEMEMBER ('dbcreator'),
3>   IS_SRVROLEMEMBER ('sysadmin', 'BUILTIN\Users'),
4>   IS_SRVROLEMEMBER ('setupadmin', 'stock')
5> go

----------- ----------- -----------
          1           0           1

(1 行受影响)
MySQL
SQLite

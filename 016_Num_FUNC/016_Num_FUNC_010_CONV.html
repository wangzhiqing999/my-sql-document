进制转换 CONV (MySQL Only)

Oracle
DB2
SQL Server 


需要使用  CONVERT  函数来处理.
先将数字转换为 VARBINARY, 然后再转换为 VARCHAR.

需要注意的地方
CONVERT(VARCHAR(20),   VARBINARY,    第3个参数要写1 )



SELECT
  CONVERT(VARCHAR(20),   CONVERT(VARBINARY(2), 100),  1) A,
  CONVERT(VARCHAR(20),   CONVERT(VARBINARY(4), 1024),  1) B

A                    B
-------------------- --------------------
0x0064               0x00000400

(1 行受影响)
MySQL 


mysql> SELECT CONV  (255, 10, 2) A
    -> //
+----------+
| A        |
+----------+
| 11111111 |
+----------+
1 row in set (0.00 sec)

mysql> SELECT CONV  (255, 10, 8) A
    -> //

| A    |
+------+
| 377  |
+------+
1 row in set (0.00 sec)


mysql> SELECT CONV  (255, 10, 16) A

+------+

+------+

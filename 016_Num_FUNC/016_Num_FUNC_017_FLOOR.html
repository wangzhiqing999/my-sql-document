数学函数

向下取整 FLOOR

Oracle 


SQL> SELECT FLOOR(-23.5), FLOOR(23.5) FROM DUAL;

FLOOR(-23.5) FLOOR(23.5)
------------ -----------
         -24          23
DB2 


db2 => SELECT  FLOOR(-23.5), FLOOR(23.5)
db2 (cont.) => FROM SYSIBM.SYSDUMMY1;

1     2
----- -----
 -24.   23.

  1 条记录已选择。
SQL Server 


1> SELECT FLOOR(-23.5), FLOOR(23.5)
2> go

----- -----
  -24    23

(1 行受影响)

MySQL 


ATAN(Y,X) , ATAN2(Y,X) 
返回两个变量X 及Y的反正切。 
它类似于 Y 或 X的反正切计算,  除非两个参数的符号均用于确定结果所在象限。 

mysql> SELECT ATAN(-2,2);
        -> -0.78539816339745
mysql> SELECT ATAN2(PI(),0);
        -> 1.5707963267949

MySQL 


LOG10(X) 
返回X的基数为10的对数。
mysql> SELECT LOG10(2);
        -> 0.30102999566398
mysql> SELECT LOG10(100);
        -> 2
mysql> SELECT LOG10(-100);
        -> NULL
LOG10(X)相当于LOG(10,X)。

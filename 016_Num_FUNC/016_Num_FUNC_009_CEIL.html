数学函数

向上取整 CEIL / CEILING

Oracle 


SQL> SELECT CEIL(18.1), CEIL(-18.1) FROM DUAL;

CEIL(18.1) CEIL(-18.1)
---------- -----------
        19         -18
DB2 


db2 => SELECT CEIL(18.1), CEIL(-18.1)
db2 (cont.) => FROM SYSIBM.SYSDUMMY1;

1     2
----- -----
  19.  -18.

  1 条记录已选择。

db2 => SELECT CEILING(18.1), CEILING(-18.1)
db2 (cont.) => FROM SYSIBM.SYSDUMMY1;

1     2
----- -----
  19.  -18.

  1 条记录已选择。
SQL Server 


1> SELECT CEILING(18.1), CEILING(-18.1)
2> go

----- -----
   19   -18

(1 行受影响)
MySQL 


mysql> SELECT CEILING(18.1), CEILING(-18.1) //
+---------------+----------------+
| CEILING(18.1) | CEILING(-18.1) |
+---------------+----------------+
|            19 |            -18 |
+---------------+----------------+
1 row in set (0.03 sec)
SQLite
PostgreSQL 



Test=# SELECT CEILING(18.1), CEILING(-18.1);
 ceiling | ceiling

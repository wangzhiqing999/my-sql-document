数学函数

余数 MOD / %

Oracle 


SQL> SELECT MOD(23,5), MOD(4,1.3) FROM DUAL;

 MOD(23,5) MOD(4,1.3)
---------- ----------
         3         .1
DB2 


db2 => SELECT
db2 (cont.) =>   MOD(23,5), MOD(4,1.3)
db2 (cont.) => FROM SYSIBM.SYSDUMMY1;

1           2
----------- -----------
          3           0

  1 条记录已选择。

注：DB2中， 余数的计算，只能计算整数。
SQL Server 


1> SELECT 23%5, 4%1.3
2> go

----------- ----
          3   .1

(1 行受影响)
MySQL 


mysql> SELECT MOD(23,5), MOD(4,1.3)

+-----------+------------+
| MOD(23,5) | MOD(4,1.3) |
+-----------+------------+
|         3 |        0.1 |

mysql> SELECT 23%5, 4%1.3;

|    3 |   0.1 |
+------+-------+
1 row in set (0.00 sec)
SQLite 


sqlite> SELECT 23%5 A, 4%1.3 B;
A|B
3|0.0


注：SQLite 中， 余数的计算，只能计算整数。

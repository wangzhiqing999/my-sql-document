MySQL 


LN(X) 
返回X 的自然对数,即, X 相对于基数e 的对数。
mysql> SELECT LN(2);
        -> 0.69314718055995
mysql> SELECT LN(-2);
        -> NULL
这个函数同LOG(X)具有相同意义。

FORMAT(X,D) 
将number X设置为格式 '#,###,###.##', 以四舍五入的方式保留到小数点后D位, 而返回结果为一个字符串。

mysql> SELECT FORMAT (-123.456,2) A,
    ->        FORMAT (-123.456,1) B,
    ->        FORMAT (-123.456,0) C,
    ->        FORMAT (-123.456,-1) D
    -> //
+---------+--------+------+------+
| A       | B      | C    | D    |
+---------+--------+------+------+
| -123.46 | -123.5 | -123 | -123 |
+---------+--------+------+------+
1 row in set (0.00 sec)

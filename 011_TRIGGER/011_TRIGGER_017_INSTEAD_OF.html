INSTEAD OF 触发器

参阅 可更新视图的章节
Oracle
DB2
SQL Server 


对于一个表，有多个触发器，其中有一个是 INSTEAD OF 的情况下。
其他的触发器将失去作用。
例如：

CREATE TRIGGER InsertTest
  ON test_sub
  FOR INSERT
AS
BEGIN
  PRINT('INSERT test_sub!');
END;
Go

CREATE TRIGGER InsertTest2
  ON test_sub
  INSTEAD OF INSERT
AS
BEGIN
  PRINT('INSERT test_sub! (INSTEAD OF ) ');
END;
Go

执行
INSERT INTO test_sub VALUES( 3,1,'100');


将只会看到

INSERT test_sub! (INSTEAD OF ) 

(1 行受影响)

且数据库中没有数据被插入。

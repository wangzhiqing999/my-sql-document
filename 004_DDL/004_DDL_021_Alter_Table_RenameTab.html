修改表 - 变更表名

Oracle 


SQL> ALTER TABLE test_tab
  2    RENAME TO test_tab2;

Table altered.




注：
Oracle 10 中， 可以简单的使用  RENAME old_name  TO new_name; 语句来处理！

SQL> CREATE TABLE test_tab (
  2    id   INT,
  3    name VARCHAR(10),
  4    age  INT,
  5    val  VARCHAR(10)
  6  );

表已创建。

SQL> 
SQL> 
SQL> RENAME test_tab  TO test_tab2; 

表已重命名。


根据文档显示，可以使用 RENAME ， 对 表、视图、序列号、私有同义词 进行更名的处理。
Use the RENAME statement to rename a table, view, sequence, or private synonym.

注意事项：
The object must be in your own schema.
DB2 


db2 => RENAME TABLE  test_tab TO test_tab2;
DB20000I  SQL 命令成功完成。
SQL Server 


1> EXEC sp_rename  N'test_tab',  N'test_tab2';
2> go
注意: 更改对象名的任一部分都可能会破坏脚本和存储过程。
MySQL 


mysql> ALTER TABLE test_tab
    ->   RENAME TO test_tab2;
    -> //
Query OK, 0 rows affected (0.03 sec)
SQLite 


sqlite> ALTER TABLE test_tab
   ...>         RENAME TO test_tab2;
sqlite>
PostgreSQL 


Test=# ALTER TABLE test_tab
Test-#   RENAME TO test_tab2;

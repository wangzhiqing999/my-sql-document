MySQL 




修改当前登录用户的密码：  （也就是修改自己的密码）

mysql> SET PASSWORD = PASSWORD('test');
Query OK, 0 rows affected (0.00 sec)

要想更改已有账户的密码，使用UPDATE来设置Password列值：

shell> mysql -u root mysql

       -> WHERE Host = '%' AND User = 'francis';

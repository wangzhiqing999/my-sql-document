对用户分配角色

Oracle 


SQL> GRANT CONNECT TO test_user;

Grant succeeded.

SQL> GRANT DBA TO test_user;

Grant succeeded.
DB2
SQL Server 


1> -- 通过将 test_user 加入数据库角色 db_owner
2> exec sp_addrolemember 'db_owner', 'test_user'
3> go
MySQL 


mysql> GRANT ALL PRIVILEGES ON *.* TO 'test_user'@'localhost'
    ->   IDENTIFIED BY 'testpassword' WITH GRANT OPTION;
    -> //
Query OK, 0 rows affected (0.00 sec)

mysql> GRANT ALL PRIVILEGES ON *.* TO 'test_user'@'%';
    -> //
Query OK, 0 rows affected (0.00 sec)

MySQL 


REPEAT(str,count) 
返回一个由重复的字符串str 组成的字符串，字符串str的数目等于count 。 
若 count <= 0,则返回一个空字符串。若str 或 count 为 NULL，则返回 NULL 。

mysql> SELECT REPEAT('MySQL', 3);
        -> 'MySQLMySQLMySQL'

mysql> SELECT REPEAT ('ABC', 3) //
+-------------------+
| REPEAT ('ABC', 3) |
+-------------------+
| ABCABCABC         |
+-------------------+
1 row in set (0.00 sec)

字符函数

去除右边空格与字符RTRIM

Oracle 


可以去除空格
或者指定字符
或者指定字符集合



SQL> SELECT RTRIM('   Ora   ') A,
  2         RTRIM('***Ora***', '*') B,
  3         RTRIM('zyxOraxyz', 'xyz') C
  4  FROM
  5    dual;

A            B            C
------------ ------------ ------------
   Ora       ***Ora       zyxOra
DB2 


db2 => SELECT RTRIM('   DB2   ') A
db2 (cont.) => FROM SYSIBM.SYSDUMMY1;

A
---------
   DB2

  1 条记录已选择。
SQL Server 


仅仅能去除空格

1> SELECT RTRIM('   SQL   ') A
2> go
A
---------
   SQL

(1 行受影响)
MySQL 


mysql> SELECT RTRIM('   SQL   ') A //
+--------+
| A      |
+--------+
|    SQL |
+--------+
1 row in set (0.00 sec)
SQLite 

与 Oracle 处理机制相同

SELECT RTRIM('   Ora   ') A,
	   RTRIM('***Ora***', '*') B,
	   RTRIM('zyxOraxyz', 'xyz') C;

A|B|C
   Ora|***Ora|zyxOra

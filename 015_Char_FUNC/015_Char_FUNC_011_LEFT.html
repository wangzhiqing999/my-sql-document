字符函数

获取字符串左边开始指定个数的字符 LEFT

Oracle 


没有这个函数
但是可以通过 SUBSTR 实现

SQL> SELECT
  2    SUBSTR('abcdefg', 1, 3)
  3  FROM
  4    dual;

SUB
---
abc
DB2 


db2 => SELECT  LEFT('abcdefg', 3)
db2 (cont.) => FROM SYSIBM.SYSDUMMY1;

1
-------
abc

  1 条记录已选择。
SQL Server 


1> SELECT LEFT('abcdefg', 3)
2> go

------
abc

(1 行受影响)
MySQL 


mysql> SELECT LEFT('abcdefg', 3)
    -> //
+--------------------+
| LEFT('abcdefg', 3) |
+--------------------+
| abc                |
+--------------------+
1 row in set (0.00 sec)
SQLite
PostgreSQL 


Test=# SELECT  LEFT('abcdefg', 3);
 left
------
 abc

字符函数

返回由重复的空格组成的字符串 SPACE

Oracle 


没有这个函数
但是可以通过 LPAD 或者 RPAD 来实现
SQL> SELECT
  2    LPAD(' ', 10) || 'A' AS A,
  3    RPAD(' ', 10) || 'B' AS B
  4  FROM 
  5    dual;

A           B
----------- -----------
          A           B
DB2 


SELECT 'A' || SPACE(10) || 'A' 
FROM SYSIBM.SYSDUMMY1;

-------------
A          A
SQL Server 


1> SELECT SPACE(10) + 'A'
2> go

-----------
          A

(1 行受影响)
MySQL 


mysql> SELECT SPACE(10)  A //
+------------+
| A          |
+------------+
|            |
+------------+
1 row in set (0.00 sec)

mysql> SELECT SPACE(40)  A //
+------------------------------------------+
| A                                        |
+------------------------------------------+
|                                          |
+------------------------------------------+
1 row in set (0.00 sec)
SQLite
PostgreSQL 


没有这个函数
但是可以通过 LPAD 或者 RPAD 或者 repeat 来实现

Test=# SELECT
Test-#   LPAD(' ', 10) || 'A' AS A,
Test-#   RPAD(' ', 10) || 'B' AS B,
Test-#   repeat(' ', 10) || 'C' AS C;
      a      |      b      |      c
-------------+-------------+-------------
           A |           B |           C
(1 行记录)

字符函数

由字符获取ASCII码ASCII

Oracle 


SQL> SELECT ASCII('D'), ASCII('Dave') FROM dual;

ASCII('D') ASCII('DAVE')
---------- -------------
        68            68
DB2 


db2 => SELECT ASCII('D'), ASCII('Dave') FROM SYSIBM.SYSDUMMY1;

1           2
----------- -----------
         68          68

  1 条记录已选择。
SQL Server 


1> SELECT ASCII('D'), ASCII('Dave')
2> go

----------- -----------
         68          68

(1 行受影响)
MySQL 


mysql> SELECT ASCII('D'), ASCII('Dave');
+------------+---------------+
| ASCII('D') | ASCII('Dave') |
+------------+---------------+
|         68 |            68 |
+------------+---------------+
1 row in set (0.00 sec)
SQLite
PostgreSQL 


Test=#  SELECT ASCII('D'), ASCII('Dave') ;
 ascii | ascii

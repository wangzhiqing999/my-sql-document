字符函数

字符串连接 CONCAT || / +

Oracle 


SQL> SELECT
  2    CONCAT('Good ', 'Morning!') A,
  3    'Good ' || 'Morning!'  B
  4  FROM
  5    dual;

A                          B
-------------------------- --------------------------
Good Morning!              Good Morning!
DB2 


和 Oracle 一样。

db2 => SELECT
db2 (cont.) =>   CONCAT('Good ', 'Morning!') A,
db2 (cont.) =>   'Good ' || 'Morning!'  B
db2 (cont.) => FROM
db2 (cont.) =>   SYSIBM.SYSDUMMY1;

A             B
------------- -------------
Good Morning! Good Morning!

  1 条记录已选择。
SQL Server 


1> SELECT 'Good ' + 'Morning!'  B
2> go
B
-------------
Good Morning!

(1 行受影响)
MySQL 


mysql> select CONCAT('Good ', 'Morning!') A
    -> //
+---------------+
| A             |
+---------------+
| Good Morning! |
+---------------+
1 row in set (0.02 sec)


-- CONCAT_WS 第一个参数为分隔符号


mysql> SELECT CONCAT_WS(',',
    ->   'First name',
    ->   'Second name',
    ->   'Last Name') A;
+----------------------------------+
| A                                |
+----------------------------------+
| First name,Second name,Last Name |
+----------------------------------+
1 row in set (0.00 sec)

字符函数

字符串长度 LENGTH / LEN

Oracle 


SELECT 
  LENGTH('oracle SQL server') A,
  LENGTH( N'oracle SQL server') NA,
  LENGTH('中文字符') B,
  LENGTH( N'中文字符') NB,
  LENGTH('中英混合Englisg') C,
  LENGTH( N'中英混合Englisg') NC
FROM
  dual;

  A         NA          B         NB          C         NC
--- ---------- ---------- ---------- ---------- ----------
 17         17          4          4         11         11

字符函数

右边字符填充 RPAD

Oracle 


SQL> SELECT RPAD('Ora', 8) A,
  2         RPAD('Ora', 8, '*') B,
  3         RPAD('Ora', 8, 'xyz') C
  4  FROM
  5    dual;

A                B                C
---------------- ---------------- ----------------
Ora              Ora*****         Oraxyzxy
DB2
SQL Server 


没有这个函数
但是 单个字符的，可以通过REPLICATE 与 LEFT的组合来实现
多个字符的，处理起来，略为复杂。

1> SELECT
2>   LEFT('Ora' + REPLICATE(' ', 8), 8) AS A,
3>   LEFT('Ora' + REPLICATE('*', 8), 8) AS B,
4>   'Ora'
5>     + LEFT(REPLICATE('xyz', 8), 8 - Len('Ora'))
6>      AS C
7> go
A           B           C
----------- ----------- -------------
Ora         Ora*****    Oraxyzxy

(1 行受影响)

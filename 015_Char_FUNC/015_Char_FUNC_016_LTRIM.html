字符函数

去除左边空格与字符 LTRIM

Oracle 


可以去除空格
或者指定字符
或者指定字符集合


SQL> SELECT LTRIM('   Ora   ') A,
  2         LTRIM('***Ora***', '*') B,
  3         LTRIM('zyxOraxyz', 'xyz') C
  4  FROM
  5    dual;

A            B            C
------------ ------------ ------------
Ora          Ora***       Oraxyz
DB2 


db2 => SELECT
db2 (cont.) =>   LTRIM('   DB2   ')
db2 (cont.) => FROM SYSIBM.SYSDUMMY1;

1
---------
DB2

  1 条记录已选择。
SQL Server 


仅仅能去除空格



1> SELECT LTRIM('   SQL   ') A
2> go
A
---------
SQL

(1 行受影响)

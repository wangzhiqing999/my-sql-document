获取架构名称 SCHEMA_ID

Oracle
DB2
SQL Server 


SCHEMA_ID( [ schema_name ] ) 
参数
schema_name 
架构的名称。schema_name 的数据类型为 sysname。
如果未指定 schema_name，则 SCHEMA_ID 将返回调用方的默认架构的 ID。

1> SELECT SCHEMA_ID(), SCHEMA_ID('Work'), SCHEMA_ID('Report')
2> go

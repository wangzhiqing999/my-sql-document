返回列名 COL_NAME

Oracle
DB2
SQL Server 



COL_NAME ( table_id , column_id )

参数
table_id 
包含列的表的标识号。table_id 的类型为 int。

column_id 
列的标识号。column_id 参数的类型为 int。


1> SELECT
2>   COL_NAME(OBJECT_ID('t1'), 1),
3>   COL_NAME(OBJECT_ID('t1'), 2),
4>   COL_NAME(OBJECT_ID('t1'), 3),
5>   COL_NAME(OBJECT_ID('t1'), 4)
6> GO

id  c1  c2  c3

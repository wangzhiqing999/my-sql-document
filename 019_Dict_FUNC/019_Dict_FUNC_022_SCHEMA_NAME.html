SCHEMA_NAME ( [ schema_id ] )
schema_id
 架构的 ID。schema_id 的数据类型为 int。如果没有定义 schema_id，
 则 SCHEMA_NAME 将返回调用方的默认架构的名称。 

1> SELECT SCHEMA_NAME(), SCHEMA_NAME(5), SCHEMA_NAME(6)

数据类型

二进制类型

Oracle 


RAW	变长二进制数据，最大2000字节
LONG RAW 最大2G二进制数据 建议改用 BLOB数据类型
BLOB	Oracle 9i 及以前，最大4G二进制数据 Oracle10g 最大4G*数据库块大小的数据
DB2
SQL Server 


binary	定长的二进制数据，最大8000字节
varbinary	变长的二进制数据，最大8000字节 
			SQL Server2005后 可以通过 varbinary(MAX)来允许最大存储2G的数据
image	变长的二进制数据，最大2G

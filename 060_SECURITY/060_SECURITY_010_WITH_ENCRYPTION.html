安全

存储过程加密

Oracle
DB2
SQL Server 


主要使用  with encryption 来完成.

/***************创建未加密的存储过程*******************/
Create Procedure test_Original
AS
PRINT 'Hello World!';
go


/***************创建加密的存储过程*******************/
Create Procedure test_Encryption 
with encryption
AS
PRINT 'Hello World 2!';
go





EXEC sp_helptext test_Original
GO


/***************创建未加密的存储过程*******************/
Create Procedure test_Original
AS
PRINT 'Hello World!';




EXEC sp_helptext test_Encryption
GO
对象 'test_Encryption' 的文本已加密。



with encryption  功能，仅仅是不让别人简单看到你的存储过程源代码。
但此时，存储过程仍然能被execute、alter和drop。



注意：加密存储过程前应该备份原始存储过程，且加密应该在部署到生产环境前完成。

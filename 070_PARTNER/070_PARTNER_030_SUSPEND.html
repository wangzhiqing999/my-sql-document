-- 暂停数据库镜像会话
-- 虚拟机上面执行. （在本机上面执行也可以）
USE master; 
ALTER DATABASE TestSystem SET PARTNER SUSPEND 
GO

在 SQL Server Management Studio 中观察。（需要刷新一下）
本机数据库 显示为：   TestSystem(主体，挂起)
虚拟机数据库 显示为： TestSystem(镜像，挂起/正在还原...)


此时， 本机数据库  可以正常执行 SQL 操作.



虚拟机数据库由于是处于镜像的，因此数据库下拉列表中，看不到 TestSystem。

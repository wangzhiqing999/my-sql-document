数据库审计

安装配置

Oracle 



-- 查询数据库是否启用审计.
SQL> show parameter audit_trail

NAME                                 TYPE
------------------------------------ ----------------------
VALUE
------------------------------
audit_trail                          string
NONE


-- 设置启用审计.
SQL> alter system set audit_trail=DB scope=spfile;
系统已更改。


-- 然后重新启动数据库。

原因：
SQL> alter system set audit_trail=DB scope=both;
